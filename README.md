# Summary

Basic discord chat bot a lot of different functionality around API consumption.

# Installation

Installation is straight forward, the program can be built withoout dlls but some functionality will dissapear. 

## Programs needed (install on running machine)
* Git
* FFmpeg (for the music module)
* Sqlite DbBrowser

## DLLs needed (put in dependencies folder)
* Libsodium.dll
* Opus.dll

Publish with dotnet publish -c Release -r linux-arm
